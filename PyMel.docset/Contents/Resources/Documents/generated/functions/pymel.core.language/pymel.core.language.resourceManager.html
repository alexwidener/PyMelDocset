pymel.core.language.resourceManager¶

resourceManager(*args, **kwargs)¶

List resources matching certain properties.

Flags:

Long Name / Short Name Argument Types Properties
nameFilter / nf unicode
 
List only resources matching the name. Argument may contain ? and * characters.
saveAs / s unicode, unicode
 

Saves a copy of the resource (first parameter) as a separate file (second parameter). Flag can have multiple arguments, passed either as a tuple or a list.

Derived from mel command maya.cmds.resourceManager

Example:

import pymel.core as pm

# List all resources starting with an "a"
pm.resourceManager(nameFilter="a*")
