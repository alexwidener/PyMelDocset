pymel.core.system.allNodeTypes¶

allNodeTypes(*args, **kwargs)¶

This command returns a list containing the type names of every kind of creatable node registered with the system. Note that some node types are abstract and cannot be created. These will not show up on this list. (e.g. transform and polyShape both inherit from dagObject, but dagObject cannot be created directly so it will not appear on this list.)

Flags:

Long Name / Short Name Argument Types Properties
includeAbstract / ia bool
 

Show every node type, even the abstract ones which cannot be created via the ‘createNode’ command. These will have the suffix (abstract)appended to them in the list. Flag can have multiple arguments, passed either as a tuple or a list.

Derived from mel command maya.cmds.allNodeTypes
